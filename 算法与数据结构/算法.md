#### 冒泡排序

遍历数组，比较相邻的两个数，把大的数往后移这样，没遍历一次数组，最大的数就会被放在数组末端
依次减小遍历的长度，直到减少到0,最终就会形成一个升序排列的数组，反过来操作就会完成降序排列,下图来自菜鸟教程

![bubbleSort](.\res\bubbleSort.gif)

**注意**此处的第几遍并不是程序真正执行时外层循环执行的次数
原数组；2，7，7，8，3，2，9，1，4，9

第一遍：2，7，7，3，2，8，1，4，9，9

第二遍：2，7，3，2，7，1，4，8，9，9

第三遍：2，3，2，7，1，4，7，8，9，9

第四遍：2，2，3，1，4，7，7，8，9，9

第五遍：2，2，1，3，4，7，7，8，9，9

第六编：2，1，2，3，4，7，7，8，9，9

第七遍：1，2，2，3，4，7，7，8，9，9

> 真正外层循环执行len(list)-1次

```python
def bubble_sort(items,comp=lambda x,y:x>y):
	for i in range(len(items) - 1):
		swapped = False
		for j in range(len(items) - i - 1):
			if comp(items[j],items[j+1]:
				items[j],items[j+1] = items[j+1],items[j]
				swapped = True
		if not swapped:
			break
	return items
# 测试代码
import random
my_list = list(range(24))
random.shuffle(my_list)
print(my_list)
sorted = bubble_sort(my_list)
print(sorted)
```

#### 搅拌算法.md
搅拌算法是冒泡算法的改进版，也可以叫做“鸡尾酒排序”“定向冒泡排序”“涟漪排序”“来回排序”“快乐小时排序”

![鸡尾酒排序](.\res\鸡尾酒排序.gif)

​																			此图来自维基百科

冒泡算法每遍历一次数组，只将最大数放在数组的最后端，搅拌排序能将最小数放在最前面，最大数放在最后面。就像是个环形，将大数放在前面之后再把小数放在前面，如此往复，最终完成排序。
为什么能提高性能？
搅拌排序依然是两层循环，内部将**大数往后**移的循环和把**小数往前**移的循环交叉执行
鸡尾酒排序等于是冒泡排序的轻微变形。不同的地方在于从低到高然后从高到低，而冒泡排序则仅从低到高去比较序列里的每个元素。他可以得到比冒泡排序稍微好一点的性能，原因是冒泡排序只从一个方向进行比对（由低到高），每次循环只移动一个项目。

以序列（2,3,4,5,1）为例，鸡尾酒排序只需要访问一次序列就可以完成排序，但如果使用冒泡排序则需要四次。但是在随机数序列的状态下，鸡尾酒排序与冒泡排序的效率与其他众多排序算法相比均比较低。

```python
def swap(items,i,j):
	items[i],items[j] = items[j],items[i]
    
def double_bubble_sort(items,comp=lambda x,y:x>y):
	for i in range(len(items) - 1):
		swapped = False
		for j in range(len(items)-1-i):
			if comp(items[j],items[j+1]):
				swap(items,j,j+1)
				swapped = True
		if swapped:
			swapped = True
			for j in range(len(items)-2-i,i,-1):
				if comp(items[j-1],items[j]):
					swap(items,j,j-1)
					swapped = True
		if not swapped:
			break
	return items
# 测试代码
import random
my_list = list(range(24))
random.shuffle(my_list)
print(my_list)
sorted = double_bubble_sort(my_list)
print(sorted)       
```
